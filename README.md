-Card : Poster Movie and TV
-Modal : Info for Movie and TV Series when card clicked
-Navbar : Menu, search bar, and sign in.
-Breadcrumbs : Breadcrumb in tv show, movie, and genres page.
-Form : Contact Us in footer.
