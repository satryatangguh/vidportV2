# Vidport
Vidport is mini project from Dibimbing and movie database website. This project create by using HTML, CSS, and Bootstrap

## Features of this App

- Home Page
- TV Show Page
- Movie Page
- Movie and TV Show Card View
- Detail of Movie and TV Show

## Card Product Source:
- Disney Hotstar ID
